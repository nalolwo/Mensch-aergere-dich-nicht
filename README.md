# Mensch ärgere dich nicht 🎲

Ein einfaches Python-Projekt zur Umsetzung des klassischen Brettspiels **„Mensch ärgere dich nicht“**.

## 🚀 Starten des Programms

Die zentrale Datei zum Ausführen des Spiels ist:

```
Startfenster.py
```

Starte das Programm einfach über die Konsole oder durch Doppelklick (je nach Systemkonfiguration):

```
python Startfenster.py
```

## ⚠️ Wichtiger Hinweis zur Kompatibilität

Dieses Projekt wurde hauptsächlich unter **Windows** entwickelt und getestet.
Es kann daher sein, dass das Programm auf anderen Betriebssystemen (z. B. Linux oder macOS) **nicht korrekt funktioniert** oder Anpassungen erforderlich sind.

## 📦 Voraussetzungen

* Benötigt Python 3 und tkinter (muss unter Linux möglicherweise separat installiert werden).


## 🛠️ Hinweise

* Stelle sicher, dass sich alle benötigten Dateien im selben Verzeichnis befinden.
* Falls Fehler auftreten, überprüfe Pfade und installierte Python-Version.

## 📄 Lizenz

Dieses Projekt dient zu Lern- und Demonstrationszwecken.
